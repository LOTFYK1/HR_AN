# Employee Overview

## Description
This project contains a Power BI report summarizing employee data. It includes Age, Department, Salary, Years at Company, and Attrition status.

## Files
- `EmployeeOverview.pbix` : Power BI report file
- `clean_data.py` : Python script for cleaning the data (optional)
- `screenshots/` : Screenshots of the report pages (optional)
- `data/employees_sample.csv` : Sample data (optional)
